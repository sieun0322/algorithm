# Step 1: Problem Interpretation

## 1. Problem Restatement

4x4 보드에서 피보나치 수가 있고, 주어진 방향으로 밀면:
- 숫자들이 해당 방향 끝까지 이동
- **연속된 두 피보나치 수**가 만나면 합쳐져서 다음 피보나치 수가 됨
- 합쳐지는 순서는 밀리는 방향의 **반대쪽**부터 처리
- 한 턴에 이미 합쳐진 숫자는 다시 합쳐지지 않음

## 2. Input Format

- `board`: 4x4 정수 배열 (피보나치 수, 0은 빈칸)
- `direction`: "up", "down", "left", "right"

## 3. Output Format

- 4x4 정수 배열 (밀기 후 상태)

## 4. Constraints

- 보드 크기: 4x4 고정
- 값: 피보나치 수 (< 2^16)
- 0 = 빈칸

## 5. Pitfalls

- **연속 피보나치**: 1+1=2, 1+2=3, 2+3=5 (합이 아니라 다음 피보나치)
- **1, 1 특수 케이스**: 둘 다 1이면 합쳐져서 2
- **합침 순서**: right로 밀면 오른쪽부터 처리 (backward = 왼쪽부터 보면서 오른쪽으로 합침)
- **이중 합침 방지**: [1,1,2,0] right → [0,0,1,3] (1+1=2, 2+2=X, 이미 합쳐진 2는 다시 못 합침)
- **빈 칸으로 이동**: 합쳐져서 생긴 빈 칸으로 뒤의 숫자가 이동 가능
