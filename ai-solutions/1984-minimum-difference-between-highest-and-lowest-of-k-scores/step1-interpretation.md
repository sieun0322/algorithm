# Step 1: Problem Interpretation

## 1. Problem Restatement

학생들의 점수가 담긴 배열이 주어진다. 이 중에서 k명의 학생을 선택해야 한다. 선택한 k명의 점수 중 최고점과 최저점의 차이가 최소가 되도록 학생들을 선택하고, 그 최소 차이값을 반환해야 한다.

## 2. Input Format

- `nums`: 정수 배열 (0-indexed), 각 원소는 학생의 점수
- `k`: 선택해야 할 학생 수 (정수)

## 3. Output Format

- 선택한 k명의 점수 중 (최고점 - 최저점)의 최소값 (정수)

## 4. Constraints

- `1 <= k <= nums.length <= 1000`
- `0 <= nums[i] <= 10^5`
- k는 항상 배열 길이 이하이므로 k명을 선택하는 것이 항상 가능

## 5. Pitfalls

- **k = 1인 경우**: 한 명만 선택하면 최고점 = 최저점이므로 차이는 항상 0
- **k = nums.length인 경우**: 모든 학생을 선택해야 하므로 전체 배열의 max - min이 답
- **중복 점수**: 같은 점수를 가진 학생이 여러 명 있을 수 있음
- **순서 무관**: 어떤 학생을 선택하든 상관없이 점수의 차이만 중요
